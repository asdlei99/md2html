# md2html

Convert Markdown to HTML

## Feature

* Search recursively in a sub directory

* Convert `README.md` to both `README.html` and `index.html`

## Installation

```
composer install
```

## Usage

```
php md2html.php .
```

```
php md2html.php tests/
```
